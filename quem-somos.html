Quem somos?

Objetivo:

Apresentar informações e novos conceitos sobre Sustentabilidade em especial o lixo eletrônico, tema escolhido para este projeto. E também mostrar as habilidades desenvolvidas durante quatro anos de curso técnico de informática para internet.

A turma do Curso Técnico de Informática para Internet - 129, que concluí o último ano letivo do curso tevee como critério para a aprovação o desenvolvimento desde website.

Para que serve este site?

Este site será usado para para fins acadêmicos, entretanto, sera usado como plataforma de informações sobre o tema propósto.

Como este site foi desenvolvido?

Este websitee usa as Linguagem de Marcação de Hipertexto, o HTTML, e a Linguagem de Folha de Estilo em Cascata, as CSS. Mas antes de codificarmos este site, usados a ferramenta Mockflow para a criação do Wireframe.
